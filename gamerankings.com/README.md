##Goals
1. PlayStation 2 list of games - https://www.gamerankings.com/browse.html?site=ps2&cat=0&year=0&numrev=3&sort=2&letter=&search= -

    * ID
    * Platform
    * Title
    * Main genre << Second genre << Third genre << Forth genre

    * Link
3. Output:
    * JSON
